# Engineering Concepts

## 📌 Purpose

This folder contains practice exercises for core **software engineering concepts** that every Java developer should master — algorithmic problem solving (Data Structures & Algorithms) and software design (Design Patterns).

## 📂 Contents

| Folder | Description |
|--------|-------------|
| [`Data structures and Algorithms/`](<Data structures and Algorithms/README.md>) | Search algorithms (linear/binary search) and recursion, with Big-O analysis |
| [`Design principles & Patterns/`](<Design principles & Patterns/README.md>) | Singleton and Factory Method design pattern implementations |

## 📦 Dependencies

None — all exercises in this folder are plain Java files with no external libraries. They can be compiled and run with the JDK alone (`javac` / `java`).

## ▶️ Usage

Each sub-folder contains standalone `.java` files. Compile and run them directly:

```bash
cd "Data structures and Algorithms"
javac EcommerceSearchExample.java
java EcommerceSearchExample
```

See each sub-folder's README for class-specific instructions.
