# JAVA FS Engineer

## 📌 Purpose

This folder groups all practice modules related to the **Java Full Stack Engineer** deep-skilling track. It contains the **Engineering concepts** module (foundational CS topics such as data structures, algorithms, and design patterns) and the **PROGRAMING LANGUAGES** module (logging, testing, and PL/SQL database programming).

## 📂 Contents

| Folder | Description |
|--------|-------------|
| [`Engineering concepts/`](<Engineering concepts/README.md>) | Core CS & software engineering concept practice (DSA, design patterns) |
| [`PROGRAMING LANGUAGES/`](<PROGRAMING LANGUAGES/README.md>) | PL/SQL programming, SLF4J logging, and TDD with JUnit & Mockito |

## 📦 Dependencies

None at this level — dependencies are declared per-exercise inside each sub-module.

## ▶️ Usage

Navigate into the relevant sub-folder and follow the instructions in its README.
